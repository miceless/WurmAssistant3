AldursLab.WurmAssistant3.exe can be run with additional command line params:

-WurmUnlimited 
: If present, starts WA in Wurm Unlimited mode. 
: If not present, starts in Wurm Online mode.

-RelativeDataDir 
: If present, will use a data directory relative to: this .exe if running directly, or launcher directory if started by launcher
: If not present, will use a path in %localappdata%
